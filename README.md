## 📝 소개
 
>Taskify는 가족, 회사 등 다양한 커뮤니티를 생성하고 멤버를 초대하여
일정과 할 일 목록을 함께 관리할 수 있는 웹 플랫폼입니다.

<br>

## 🎯 목적

> 업무, 프로젝트, 사내 이벤트 등을 통합적으로 관리
> 멤버 간 효율적인 소통과 일정 관리

## 📁 폴더 구조

```
src/
├── pages/         # Next.js Pages Router 기반 페이지
├── features/      # 기능(도메인) 단위로 구성된 모듈
├── components/    # 전역에서 재사용되는 공통 UI 컴포넌트
├── hooks/         # 공통 커스텀 훅
├── services/      # Axios 인스턴스 및 API 통신 로직
├── constants/     # 상수 및 공통 설정 값
├── types/         # TypeScript 타입 정의
├── utils/         # 공통 유틸 함수
├── styles/        # 전역 스타일 및 CSS Modules
└── assets/        # 이미지 및 정적 리소스
```

<br>

## 🚀 실행 방법

```
# 1. 저장소 클론
git clone https://github.com/JinHyuk-Kim66/Taskify.git

# 2. 폴더 이동
cd coworkers

# 3. 패키지 설치
npm install

# 4. 개발 서버 실행
npm run dev
```
